ТИПОГРАФИКА

Оба шрифта подключать через Google Fonts

ОТСТУПЫ
Сетка 8px, все значения кратны 8 (или 4 для мелких случаев):
4px — микро-отступы (иконка-текст)
8px — базовая единица
16px — внутри карточек
24px — между связанными элементами
32px — между группами элементов
48px — между подсекциями
64-96px — между крупными секциями

ПРИНЦИПЫ ВЁРСТКИ (bento box, издательский стиль, НЕ конструкторский)
Карточки разного размера в одной сетке — не одинаковые 3-в-ряд, а сочетания: один крупный блок + два маленьких, длинная горизонтальная плашка + квадратная рядом.

Карточки складываются друг с другом по принципу тетриса — не просто разный размер в стандартной сетке, а формы, которые стыкуются вплотную, без единообразного зазора между всеми блоками: L-образная карточка может огибать угол соседней квадратной, узкая вертикальная плашка — встраиваться в выемку между двумя горизонтальными карточками. Сетка должна читаться как единая мозаика/паззл, а не как набор отдельных одинаково расставленных блоков. Используй CSS grid с разными grid-column/grid-row span для создания таких неровных, "сцепленных" комбинаций, а не просто flex-обёртку с одинаковыми gap.

Контент не всегда строго по центру — иногда текст прижат влево, заголовок асимметрично сдвинут, фото намеренно крупнее текстового блока.

Отдельные элементы могут намеренно выходить за границы карточки (крупная цифра/слово перекрывает край).

Разная плотность по высоте секций — где-то много воздуха (одна мысль на экран), где-то плотная сетка из 5-6 карточек.

Скругления варьируются по размеру элемента, не один и тот же border-radius везде.

Акцент #BB4500 используется скачками — не равномерно по странице, а то целая секция, то просто обводка кнопки, то одно слово в заголовке.

ЧЕГО ИЗБЕГАТЬ
Признаки шаблонного конструкторского сайта (Тильда-стиль): одинаковая высота всех блоков, контент всегда по центру в одной колонке, паттерн "картинка слева/текст справа" в каждом втором блоке без вариаций, иконки в одинаковых кружочках для всех преимуществ, всё строго симметрично.

Типовые AI-дефолты: стандартные мягкие тени Tailwind (box-shadow: 0 4px 6px rgba(0,0,0,0.1)), идеально равномерная сетка 3 карточки в ряд на каждую секцию, один и тот же border-radius для всех элементов без вариаций.
